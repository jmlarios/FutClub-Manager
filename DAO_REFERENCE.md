## 1. Player Model

**Package**: `com.futclub.model`  
**Class Name**: `Player`  
**Table**: `players`

### Field Specifications

| Field Name | Java Type | Database Column | Constraints | Notes |
|------------|-----------|-----------------|-------------|-------|
| playerId | `int` | player_id | PRIMARY KEY, AUTO_INCREMENT | Generated by database |
| firstName | `String` | first_name | NOT NULL | Player's first name |
| lastName | `String` | last_name | NOT NULL | Player's last name |
| dateOfBirth | `java.sql.Date` | date_of_birth | nullable | Birth date |
| position | `String` | position | NOT NULL | Must be valid position code |
| shirtNumber | `int` | shirt_number | UNIQUE, 1-99 | Jersey number |
| status | `String` | status | NOT NULL | AVAILABLE/INJURED/SUSPENDED/UNAVAILABLE |
| overallRating | `int` | overall_rating | 1-99 | Player's overall rating |
| fitnessLevel | `int` | fitness_level | 0-100 | Current fitness percentage |
| injuryDetails | `String` | injury_details | nullable | Description of injury if applicable |
| joinedDate | `java.sql.Date` | joined_date | nullable | Date joined the club |
| contractEnd | `java.sql.Date` | contract_end | nullable | Contract expiration date |
| nationality | `String` | nationality | nullable | Player's nationality |
| heightCm | `int` | height_cm | nullable | Height in centimeters |
| weightKg | `int` | weight_kg | nullable | Weight in kilograms |
| preferredFoot | `String` | preferred_foot | nullable | LEFT/RIGHT/BOTH |

### Valid Position Codes
- `GK` - Goalkeeper
- `CB` - Center Back
- `LB` - Left Back
- `RB` - Right Back
- `CDM` - Central Defensive Midfielder
- `CM` - Central Midfielder
- `CAM` - Central Attacking Midfielder
- `LW` - Left Winger
- `RW` - Right Winger
- `ST` - Striker

### Valid Status Values
- `AVAILABLE` - Available for selection
- `INJURED` - Currently injured
- `SUSPENDED` - Suspended from playing
- `UNAVAILABLE` - Unavailable for other reasons

## 9. Usage Examples

### Example 1: Creating a New Player

```java
// Create a new player
Player player = new Player();
player.setFirstName("Lionel");
player.setLastName("Messi");
player.setDateOfBirth(Date.valueOf("1987-06-24"));
player.setPosition("RW");
player.setShirtNumber(10);
player.setStatus("AVAILABLE");
player.setOverallRating(93);
player.setFitnessLevel(95);
player.setNationality("Argentina");
player.setHeightCm(170);
player.setWeightKg(72);
player.setPreferredFoot("LEFT");

// Insert using DAO
PlayerDAO playerDAO = new PlayerDAOImpl();
playerDAO.insert(player);

// player.getPlayerId() now contains the auto-generated ID
System.out.println("Created player with ID: " + player.getPlayerId());
```

### Example 2: Updating Player Status

```java
PlayerDAO playerDAO = new PlayerDAOImpl();

// Get player
Player player = playerDAO.getById(5);

// Update status
player.setStatus("INJURED");
player.setInjuryDetails("Hamstring strain, expected recovery 2 weeks");
player.setFitnessLevel(30);

// Save changes
playerDAO.update(player);
```

### Example 3: Recording Match Result with Stats

```java
// Update match
MatchDAO matchDAO = new MatchDAOImpl();
Match match = matchDAO.getById(10);
match.setGoalsFor(3);
match.setGoalsAgainst(1);
match.setMatchStatus("COMPLETED");
matchDAO.update(match);

// Record player stats
PlayerMatchStatsDAO statsDAO = new PlayerMatchStatsDAOImpl();

stats.setPlayerId(5);
stats.setMatchId(10);
stats.setMinutesPlayed(90);
stats.setGoals(2);
stats.setAssists(1);
stats.setRating(9.5);
stats.setShots(6);
stats.setShotsOnTarget(4);
stats.setWasStarter(true);

statsDAO.insert(stats);
```

## Important Reminders

### ⚠️ CRITICAL NAMING RULES

1. **Field names MUST match exactly** - Case-sensitive!
2. **Data types MUST match exactly** - `java.sql.Date` vs `java.util.Date` matters!
3. **Getter/setter names follow JavaBean convention**:
   - Getter: `getFieldName()` (or `isFieldName()` for boolean)
   - Setter: `setFieldName(Type value)`
4. **Primary keys use singular**: `playerId`, NOT `player_id`
5. **Foreign keys use singular**: `userId`, NOT `users_id`
6. **Boolean fields use `is` prefix**: `isActive`, NOT `getActive`

### Data Type Mappings

| Database Type | Java Type |
|--------------|-----------|
| INTEGER | `int` |
| TEXT | `String` |
| DATE | `java.sql.Date` |
| TIMESTAMP | `java.sql.Timestamp` |
| REAL | `double` |
| BOOLEAN (0/1) | `boolean` |

## Validation Recommendations

Add validation to your Model classes:

```java
public void setShirtNumber(int shirtNumber) {
    if (shirtNumber < 1 || shirtNumber > 99) {
        throw new IllegalArgumentException("Shirt number must be between 1 and 99");
    }
    this.shirtNumber = shirtNumber;
}

public void setPosition(String position) {
    String[] validPositions = {"GK", "CB", "LB", "RB", "CDM", "CM", "CAM", "LW", "RW", "ST"};
    if (!Arrays.asList(validPositions).contains(position)) {
        throw new IllegalArgumentException("Invalid position: " + position);
    }
    this.position = position;
}
```

## Testing Your Models

Once you create the Model classes, test them with the existing DAOs:

```java
@Test
public void testPlayerCRUD() {
    PlayerDAO playerDAO = new PlayerDAOImpl();
    
    // Create
    Player player = new Player();
    player.setFirstName("Test");
    player.setLastName("Player");
    player.setPosition("ST");
    player.setShirtNumber(99);
    player.setStatus("AVAILABLE");
    playerDAO.insert(player);
    
    int playerId = player.getPlayerId();
    assertTrue(playerId > 0);
    
    // Read
    Player retrieved = playerDAO.getById(playerId);
    assertEquals("Test", retrieved.getFirstName());
    
    // Update
    retrieved.setStatus("INJURED");
    playerDAO.update(retrieved);
    
    Player updated = playerDAO.getById(playerId);
    assertEquals("INJURED", updated.getStatus());
    
    // Delete
    playerDAO.delete(playerId);
    assertNull(playerDAO.getById(playerId));
}
```
